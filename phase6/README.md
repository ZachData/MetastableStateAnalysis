# Phase 6 — Backwards Tracing

**Status:** not yet started

## Goal
Train tuned lens probes per layer.  Apply to cluster centroids at
each metastable window.  Unembedding gives token probability
distributions — what each cluster means semantically at each depth.

## Falsification criterion
If tuned lens distributions are incoherent at early layers,
backwards tracing isn't reliable enough to interpret.
